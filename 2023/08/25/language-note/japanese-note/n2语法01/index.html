# 「とき・～直後に」に関連する表現

# 「とき」に関連する表現

# ～際（は）／際に（は）

動辞書形・た形 / 名ーの＋際（は）/ 際に（は）

在…… 时候”“借…… 时机”。语气比「〜とき」更书面语，并且带有 “时机，契机” 的意思。

  • 外出する際には、必ず鍵をかけてください。
  • お近くにお越しの際は、ぜひお立ち寄りください。
  • 先月帰省した際に、高校の先生を訪(たず) ねた。

# ～際・～際は・～際に

～際は：表示强调

～際に：表示在其事件内采取某种动作或发生某种事情

  • アメリカの大統領は来日した際に、私たちの大学でスピーチを行った。
